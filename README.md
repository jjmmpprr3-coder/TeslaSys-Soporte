# TeslaSys Chile - Sistema de Registro de Solicitudes de Soporte

Aplicación web interactiva desarrollada para la gestión y registro de solicitudes de soporte técnico de los colaboradores de **TeslaSys Chile**. Proyecto correspondiente a la Evaluación Sumativa 2 de la asignatura **Desarrollo Frontend** en INACAP.

---

## 🚀 Características del Proyecto

* **Interfaz Responsiva:** Diseño estructurado en dos columnas (Formulario de registro y Panel de visualización) adaptable a dispositivos móviles mediante CSS Flexbox.
* **Validación de Datos en Tiempo Real:** Control estricto del DOM para asegurar que ningún campo quede vacío y que las direcciones de correo electrónico contengan el carácter obligatorio `@`.
* **Persistencia Local:** Integración de `localStorage` para garantizar que la información se almacene de forma permanente en el navegador y no se pierda al recargar la página.
* **Componentes Visuales Dinámicos:** Generación iterativa de tablas de datos con estados vacíos interactivos y etiquetas (badges) de colores personalizadas según el tipo de solicitud (Hardware, Software, Redes y Seguridad).

---

## 🛠️ Tecnologías y Herramientas Utilizadas

* **HTML5:** Estructuración semántica del formulario y paneles contenedores.
* **CSS3:** Estilos corporativos basados en la paleta de colores de la empresa, efectos interactivos (`hover`, `active`), animaciones de desvanecimiento (`fade-in`) y efectos de tarjetas flotantes.
* **JavaScript (ES6):** Programación lógica orientada a eventos, manipulación del DOM, estructuras de control (`if/else`, `forEach`), y manejo del almacenamiento local.
* **GitHub Copilot:** Herramienta de Inteligencia Artificial utilizada de forma iterativa para el codiseño visual y optimización de animaciones complejas.

---

## 📋 Estructura de Archivos del Repositorio

* `index.html`: Estructura principal y maquetación semántica.
* `styles.css`: Estilos, animaciones y diseño responsivo.
* `script.js`: Lógica de validación, almacenamiento persistente y renderizado dinámico.
* `IA_UTILIZADA.txt`: Archivo obligatorio de justificación de prompts y resultados de IA.
* `enlace_github.txt`: Enlace de entrega del proyecto.

---

## ⚙️ Instrucciones de Ejecución Local

1. Descargue o clone este repositorio en su máquina local.
2. Asegúrese de que todos los archivos permanezcan juntos en la misma raíz de la carpeta.
3. Abra el archivo `index.html` en cualquier navegador web moderno (Google Chrome, Microsoft Edge, Mozilla Firefox, etc.).
