# TeslaSys-Soporte
Aplicación web interactiva para registrar y visualizar solicitudes de soporte técnico realizadas por colaboradores.

## Funcionalidad
- Formulario HTML para ingresar nombre, área y descripción del problema.
- Registro de solicitudes en memoria temporal (arreglo de objetos en JavaScript).
- Visualización dinámica de todas las solicitudes en pantalla usando manipulación del DOM.

## Uso
1. Abrir `index.html` en un navegador.
2. Completar el formulario y enviar.
3. Revisar la sección **Solicitudes registradas** para ver el historial de registros durante la sesión.
